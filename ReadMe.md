
# Web Scraping using BeautifulSoup

## Overview
A small Python project that demonstrates how to parse local HTML files and scrape tabular data from the web using BeautifulSoup.

## Features
- Parse local HTML documents with BeautifulSoup
- Navigate tags, parents, children, and sibling elements
- Extract table rows and cell values from HTML tables
- Scrape movie ranking data from a live archived web page
- Export scraped data to both CSV and SQLite database formats

## Installation
```bash
pip install beautifulsoup4 requests pandas
```

## Project Files
- `htmlscrape.py` parses `salary.html` and demonstrates `prettify()`, tag selection, child access, parent access, and sibling traversal.
- `tablescrape.py` parses `table.html` and extracts table rows and table cells.
- `moviescrape.py` scrapes the top 50 ranked films from an archived page and saves the results locally.
- `salary.html` is a local sample HTML file used for basic parsing practice.
- `table.html` is a local sample HTML table used for table extraction practice.
- `top_50_films.csv` is the CSV output generated by `moviescrape.py`.
- `Movies.db` is the SQLite database generated by `moviescrape.py` with the table `Top_50`.

## Requirements
- Python 3.x
- `beautifulsoup4`
- `requests`
- `pandas`

## How It Works
### 1. Parse a local HTML file
`htmlscrape.py` opens `salary.html`, builds a BeautifulSoup parser, and demonstrates:
- formatted HTML output with `prettify()`
- selecting the page title
- finding the first `h3` tag
- accessing a child `b` tag
- moving from a child to its parent
- finding all `h3` tags
- locating the next sibling of an element

### 2. Extract data from an HTML table
`tablescrape.py` opens `table.html`, parses the file, and shows how to:
- find all `tr` elements
- access the first row
- extract its `td` cell values

### 3. Scrape and store remote data
`moviescrape.py` requests an archived film-ranking page, extracts the first 50 rows from the main table, and stores:
- `Average Rank`
- `Film`
- `Year`

The script then saves the results to:
- `top_50_films.csv`
- `Movies.db` in the table `Top_50`

## Usage
Run each script from the project folder:

```bash
python htmlscrape.py
python tablescrape.py
python moviescrape.py
```

## Example
```python
from bs4 import BeautifulSoup
import requests

response = requests.get('https://example.com')
soup = BeautifulSoup(response.content, 'html.parser')
```

## Output
- Console output for HTML parsing examples in `htmlscrape.py` and `tablescrape.py`
- `top_50_films.csv` containing the scraped movie dataset
